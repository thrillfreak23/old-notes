null subject.  not just unspec but null.  unspec means there's a logical subject but a null subject is a different kind of verb like It's raining.  there is no it.  the idea of an "it" doing the raining is absurd, ergo the verb with that valency is a different kind of verb, one that doesn't have a subject.  it's just an activity happening.  full stop.

* Valencizer Tier Hierachy and Rules
** TIER1 :: the Directional Tier
hek-  :: PROVENANT
ho-   :: APPETENT
shar- :: AVERTENT
sor-  :: MANSENT
/NEW/ zath- :: TERMINATIVE – The action is performed until it is completely finished/exhausted.
/NEW/ pel-  :: INCHOATIVE – The action is just beginning or "bursting" forth.
** TIER2 :: the Social-Relative Tier
skâ-  :: EXCEDENT
vrî-  :: DEFICIENT
kash- :: OBSTANT
ban   :: BENEFICIENT
/NEW/ dôr-  :: MALEFICIENT – The opposite of ban- (Beneficient). While kash-
               (Obstant) blocks, dôr- implies the action actively harms or insults a third party.
/NEW/ mîv-  :: DEFERENTIAL – The action is performed specifically to show
               honor or lower one's status relative to the object.
** TIER3 :: the Relational Tier
jeg-  :: COOPERANT
ket-  :: THEMATIC
gez-  :: CAUSAL
/* vuk-  :: RECIPROCAL */
/* zâ-   :: REFLEXIVE  */
/NEW/ nath- :: INSTRUMENTAL – The subject uses a tool or specific method to complete the action.
/NEW/ vrok- :: CIRCUMSTANTIAL – The action is defined or restricted by the
         environment/setting (e.g., "to dance because of the rain").
** TIER4 :: the Internal-Perspective Tier
moz-  :: SIMULANT
thin- :: SUBJECTIVE
/NEW/ krel- :: CONATIVE – "To try to [Verb]." (e.g., instead of a separate
verb for "try," the valencizer shifts the focus to the attempt).
/NEW/ fesh- :: HABITUAL – The action isn't a one-off; it's a defining trait or routine of the subject.

*** jeg-  : WITH- (Cooperation/Instrument: "A does the verb alongside or using B")
*** ket-  : ABOUT- (Topic: "A does the verb concerning the subject of B")
*** gez-   : CAUS- (Causative: "A does the verb, but B is the reason it happened")
*** vuk-  : RECIPROCAL- (Comparison: Each other)
*** zâ-   : REFLEXIVE- (themself)
** the State of Mind Set (Internal/Perspective)
*** moz-  : SHAM- (Pretense: "A does the verb as an act or a lie toward B")
*** thin- : INWARD- (Subjective: "A does the verb within their own mind/opinion")
